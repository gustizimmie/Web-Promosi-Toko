Syarat dan Ketentuan

Terakhir diperbarui: 18 Maret 2026

Dengan membeli, menginstal, atau menggunakan aplikasi kasiraulia ("Layanan"), Anda setuju untuk terikat oleh syarat dan ketentuan berikut. Harap baca dengan saksama.

1. Lisensi Software

Auliasoft memberikan lisensi non-eksklusif dan tidak dapat dipindahtangankan kepada Anda untuk menggunakan software kasiraulia sesuai dengan paket yang dibeli (TKO-1 hingga TKO-7). Lisensi dibatasi oleh wilayah atau jumlah perangkat sesuai detail paket yang dipilih saat pembelian.

2. Masa Berlaku dan Aktivasi

  • Paket Lisensi (Offline/LAN) berlaku selamanya untuk satu kali bayar.
  • Layanan Cloud Add-on dan Paket Cloud murni memiliki masa berlaku (berlangganan) bulanan atau tahunan.
  • Aktivasi software memerlukan dokumen identitas dan bukti pembelian sah.

3. Dukungan Teknis

Kami memberikan dukungan teknis (support) gratis selama 18 bulan untuk paket tertentu (seperti TKO-2 s/d TKO-5). Support mencakup bantuan instalasi, penggunaan fitur, dan perbaikan bug sistem. Perpanjangan support dapat dilakukan dengan biaya tambahan setelah masa berlaku habis.

4. Batasan Penggunaan

Anda dilarang untuk:

  • Melakukan reverse engineering, decompile, atau mencoba mengambil source code aplikasi.
  • Mendistribusikan ulang atau menjual kembali software tanpa izin tertulis dari Auliasoft.
  • Menggunakan aplikasi untuk kegiatan ilegal atau melanggar hukum.

5. Batasan Tanggung Jawab

Meskipun kami berusaha memberikan software terbaik, Kami tidak bertanggung jawab atas kerugian finansial atau data karena kesalahan pengguna, kegagalan perangkat keras (hardware), atau faktor eksternal lainnya di luar kendali sistem kami.

6. Refund (Pengembalian Dana)

Dana yang sudah dibayarkan tidak dapat dikembalikan setelah proses instalasi dan aktivasi lisensi diselesaikan oleh tim teknis kami.
