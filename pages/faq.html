Pertanyaan Umum (FAQ)

Temukan jawaban cepat atas semua pertanyaan Anda tentang aplikasi kasir kasiraulia.

Paket & Harga

Apakah ada biaya bulanan untuk Paket Lisensi?

Tidak ada. Paket Lisensi (TKO-1 hingga TKO-5) adalah sistem "Sekali Bayar" untuk lisensi selamanya. Anda hanya perlu membayar biaya tambahan jika ingin berlangganan layanan Cloud Add-on tahunan.

Apa perbedaan TKO-1 dan TKO-2?

TKO-1 dikhususkan untuk UMKM mikro di satu wilayah (Desa/Kelurahan) dengan batasan maksimal 500 item barang. Sementara TKO-2 tidak memiliki batasan jumlah item barang dan mencakup dukungan gratis selama 18 bulan.

Apa yang dimaksud dengan Cloud Add-on pada TKO-5?

TKO-5 memungkinkan database offline Anda terintegrasi ke dashboard web. Owner bisa memantau laporan dari HP melalui layanan cloud tahunan:

• Dashboard Web HP Owner: Rp 300.000 / Tahun
• E-Kasir, E-Sales, E-Order, E-Beli: Rp 600.000 / Tahun

Apa perbedaan Paket Lisensi (TKO 1-5) dan Paket Cloud (TKO 6-7)?

Paket Lisensi (TKO 1-5) adalah bayar sekali, aplikasi berjalan di komputer lokal atau jaringan LAN Anda. Paket Cloud (TKO 6-7) adalah sistem sewa bulanan dimana aplikasi sepenuhnya berbasis web dan bisa diakses dari mana saja menggunakan HP, laptop, atau tablet dengan koneksi internet.

Apakah bisa upgrade paket di kemudian hari?

Ya, Anda bisa melakukan upgrade paket kapan saja. Hubungi tim kami via WhatsApp untuk informasi biaya upgrade dari paket Anda saat ini ke paket yang lebih tinggi.

Teknis & Operasional

Apakah aplikasi bisa berjalan tanpa koneksi internet?

Ya, untuk paket Lisensi Offline/LAN (TKO 1-5), aplikasi berjalan sepenuhnya secara lokal. Internet hanya dibutuhkan saat aktivasi pertama kali atau jika Anda menggunakan fitur Cloud Add-on.

Berapa komputer yang bisa menggunakan satu lisensi?

TKO-1 hingga TKO-3 adalah lisensi untuk 1 komputer. TKO-4 dan TKO-5 mendukung jaringan LAN dengan multi komputer dalam satu lokasi (jaringan lokal yang sama).

Apakah data saya aman jika komputer rusak?

Untuk paket Lisensi Offline, data tersimpan di server lokal Anda. Kami sangat menyarankan untuk melakukan backup data secara rutin. Untuk paket Cloud (TKO 6-7), data tersimpan aman di server cloud kami dengan enkripsi dan backup otomatis.

Bagaimana cara instalasi aplikasi setelah pembelian?

Setelah pembayaran dikonfirmasi, tim kami akan menghubungi Anda untuk proses instalasi dan aktivasi, baik secara remote (via TeamViewer) maupun kunjungan langsung untuk area tertentu. Proses instalasi biasanya membutuhkan waktu 30-60 menit.

Berapa lama masa support gratis yang diberikan?

Paket TKO-2 hingga TKO-5 mendapatkan support gratis selama 18 bulan sejak tanggal aktivasi. Setelah itu, Anda bisa memperpanjang support dengan biaya yang sangat terjangkau.
