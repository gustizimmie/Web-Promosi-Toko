Pertanyaan Umum (FAQ)

Pertanyaan yang paling sering ditanyakan oleh pelanggan kami

Apakah ada biaya bulanan untuk Paket Lisensi?

Tidak ada. Paket Lisensi (TKO-1 hingga TKO-5) adalah sistem "Sekali Bayar" untuk lisensi selamanya. Anda hanya perlu membayar biaya tambahan jika ingin berlangganan layanan Cloud Add-on tahunan.

Apakah aplikasi bisa berjalan tanpa koneksi internet?

Ya, untuk paket Lisensi Offline/LAN, aplikasi berjalan sepenuhnya secara lokal. Internet hanya dibutuhkan saat aktivasi pertama kali atau jika Anda menggunakan fitur Cloud Add-on.

Apa perbedaan Paket Lisensi dan Paket Full Cloud?

Paket Lisensi (TKO 1-5) adalah bayar sekali, berjalan di komputer lokal Anda. Paket Cloud (TKO 6-7) adalah sewa bulanan, berbasis web penuh dan bisa diakses dari mana saja menggunakan HP, laptop, atau tablet.
